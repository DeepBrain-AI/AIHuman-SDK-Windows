# CustomVoice

| Modifier and Type | Method and Description                    |
| ----------------- | ----------------------------------------- |
| `String`          | `getId()` 음성의 ID를 확인합니다.       |
| `String`          | `getName()` 음성의 이름을 확인합니다.     |
| `String`          | `getLanguage()` 음성의 언어를 확인합니다. |
| `String`          | `getGender()` 해당 음성의 성별을 가져옵니다. (MALE/FEMALE) |
