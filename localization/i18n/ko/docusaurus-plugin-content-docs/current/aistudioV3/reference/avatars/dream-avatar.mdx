# 드림 아바타 생성

이미지를 사용하여 드림 아바타를 생성합니다.

<br/>

## 1. API endpoint

```
POST https://app.aistudios.com/api/odin/v3/avatars/dreamavatar
```

<br/>

## 2. Request parameter

|key|desc|type|required|default|
|:---|:---|:---|:---|:---|
|url|이미지 URL|String|true|-|
|name|드림 아바타의 이름|String|true|-|
|gender|드림 아바타의 성별|String enum (MALE, FEMALE)|true|-|
|tts|드림 아바타의 TTS|Json|false|null|

<br/>

## 3. Response parameters

|key|desc|type|
|:---|:---|:---|
|avatarId|생성된 드림 아바타의 아이디|String|

<br/>
