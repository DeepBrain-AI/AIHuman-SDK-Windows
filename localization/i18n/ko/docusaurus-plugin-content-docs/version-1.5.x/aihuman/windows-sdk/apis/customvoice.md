# CustomVoice

- assembly: AIHuman.SDK.Core  
- namespace: AIHuman.Common  

| Modifier and Type | Method and Description                    |
| ----------------- | ----------------------------------------- |
| `string`          | `ID { get; }` 목소리(음성)의 아이디입니다.       |

| `string`          | `Gender { get; }` 목소리(음성)의 성별입니다. (MALE/FEMALE/UNI) 선택적으로 서비스 운영에 활용될 수 있습니다. |
