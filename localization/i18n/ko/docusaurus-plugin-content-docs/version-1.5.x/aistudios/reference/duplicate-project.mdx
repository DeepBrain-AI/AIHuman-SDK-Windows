# Duplicate Project

## 1. API endpoint

```http
https://v2.aistudios.com/api/odin/editor/project/duplicate/${key}
```

<br/>

## 2. Response parameters

|key|desc|type|
|:---|:---|:---|
|success|Progress on Request Success|Boolean|
|projectData|Project data|Json|


<br/>
