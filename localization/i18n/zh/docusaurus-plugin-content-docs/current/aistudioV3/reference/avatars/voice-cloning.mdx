# Create Voice Clone

Create a voice clone using audio.

<br/>

## 1. API endpoint

```
POST https://app.aistudios.cn/api/odin/v3/avatars/voiceclone
```

<br/>

## 2. Request parameter

|key|desc|type|required|default|
|:---|:---|:---|:---|:---|
|url|Audio URL|String|true|-|
|name|Name of the voice clone|String|true|-|
|gender|Gender of the voice clone|String enum (MALE, FEMALE)|true|-|
|locale|Language of the sample voice for the voice clone|String enum (en, ko, zh)|false|en|

<br/>

## 3. Response parameters

|key|desc|type|
|:---|:---|:---|
|voiceId|ID of the created voice clone|String|

<br/>
