# Image Generation

Enter a topic to generate an image related to that topic.

<br/>

## 1. API endpoint

```
POST https://app.aistudios.cn/api/odin/v3/tools/image-generator
```

<br/>

## 2. Request parameter

|key|desc|type|required|default|
|:---|:---|:---|:---|:---|
|topic|Topic of the image to be generated|String|true|-|
|style|Style of the image to be generated<br />(`PHOTOGRAPHIC` \| `VECTOR_IMAGE` \| `ILLUSTRATION` \| `3D`)|String|false|`PHOTOGRAPHIC`|
|size|Aspect ratio of the image to be generated<br />(`LANDSCAPE` \| `PORTRAIT` \| `SQUARE`)|String|false|`LANDSCAPE`|

<br/>

## 3. Response parameters

|key|desc|type|
|:---|:---|:---|
|downloadUrl|URL of the generated image data|String|

:::info

The generated image data can be deleted at any time, so you should download and use the image instead of using the generated URL directly.

:::

<br/>
